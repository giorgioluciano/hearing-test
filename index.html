Frequency Hearing Tester

A small web app to generate pure tones, run a frequency sweep, or play two tones to hear beats. For curiosity only — not a clinical test.

About

This page hosts a tiny Web Audio demo that generates sine tones and simple sweeps so you can explore how frequency affects perception. Use good headphones, keep volume low, and remember this is educational — not diagnostic. If you prefer, download the full standalone HTML file to run locally.
